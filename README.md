## How to use
1. configure you database name (an empty database) in application.properties
2. yarn: install node module
3. yarn start: bundle the js
4. start the web app in IDE
5. go to [http://localhost:3000](http://localhost:3000)

## How to deploy
1. yarn deploy
2. mvn install
